# Remembering Past Present Biases

## Online Data Repository
 This is the github repository of the paper "Remembering Past Present-biases" by Barna Bakó, Hubert János Kiss and Antal Ertl.

 It includes the data as well as the code of the analyses presented in the paper.

For maximum transparency, we include both pre-and post review codes.



## Data-related information

### "data" folder:

The data folder contains. the following folders: 
 - the raw data (data_raw folder)
 - the code for data cleaning (codes_data_cleaning; written in R and STATA), and
 - the cleaned data (data_cleaned), which contains the cleaned data in csv and dta.
 
 
### "codes" folder:
 
The codes folder includes are codes necessary to reproduce the results in the final text.

For clarity, we included codes both before and after the revision in the repository. Again, we thank both anonymious reviewers for providing us meaningful feedback on the manuscript.




### "output" folder:

The output folder includes all outputs -- both Figures and Tables from the codes.
